# Boston Housing ML Model
A machine learning project that predicts house prices in Boston based on features such as crime rate, number of rooms, and property age. Includes data exploration, preprocessing, model building, evaluation, and visualization of results.

### Software and Tools Requirements

1. [Github Account](https://github.com/)
2. [HerokuAccount](https://www.heroku.com/)
3. [VS Code IDE](https://code.visualstudio.com/)
4. [Git CLI](https://git-scm.com/install/)
